This website was created using Node JS to host the server. HTML and JavaScript were then used to create the functional attributes of the website. CSS was used to style the pages. The website was deployed with Heroku.

Known errors:

  • Yelp Fusion Api occasionally returns multiple of the same restaurant
  • The geolocation is not always precisely at the user location
  • Issues may arise when putting the custom coordinates in the middle of the ocean
